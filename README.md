## Supported sources and formats


JMelody supports all sources and formats supported by lavaplayer:

### Sources
 - YouTube
 - SoundCloud
 - Bandcamp
 - Vimeo
 - Twitch streams
 - Local files
 - HTTP URLs

### Formats
 - MP3
 - FLAC
 - WAV
 - Matroska/WebM (AAC, Opus or Vorbis codecs)
 - MP4/M4A (AAC codec) 
 - OGG streams (Opus, Vorbis and FLAC codecs)
 - AAC streams 
 - Stream playlists (M3U and PLS)
